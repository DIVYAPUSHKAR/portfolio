# portfolio
A modern, full-stack Portfolio Web Application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). This project showcases my skills, projects, experience, and provides a way to contact me directly through the website.

## Demo Link: https://front-pmim.onrender.com/

## Frontend
React.js
React Router
Axios
## Backend
Node.js
Express.js
Mongoose
JWT Authentication
## Database
MongoDB Atlas
## Features
Responsive and modern UI
Dynamic projects section (CRUD operations)
Admin dashboard to manage portfolio content
Contact form with email functionality
Secure authentication using JWT
